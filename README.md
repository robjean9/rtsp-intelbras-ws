# rtsp-intelbras-ws

O Sistema roda na Porta 2000, caso deseje rodar em outra porta utilize a variável de ambiente (.env): **PORT**

Variáveis de Ambiente (.env):

**DVR_IP**=192.168.X.X

**DVR_PORT**=554

**DVR_USER**=admin

**DVR_PASSWORD**=admin

**DVR_STREAM**=0 //0 -> Main Stream (HD) | 1-> Extra Stream (SD)
